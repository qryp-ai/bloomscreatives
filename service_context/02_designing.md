## Process
1. **Research & Moodboards** — audience, category, and competitor study.
   - Deep dive into brand objectives and target audience analysis
   - Competitive landscape analysis and visual audit
   - User research and empathy mapping
   - Trend analysis and design direction exploration
   - Technical requirements and constraint identification

2. **Concepting** — 3 directions; choose 1 and refine.
   - Creative concept development and visual exploration
   - Multiple design directions and style variations
   - Stakeholder feedback and collaborative refinement
   - Concept testing and user validation
   - Final direction selection and detailed development

3. **Systemize** — build the visual language and component library.
   - Design system architecture and component creation
   - Style guide development and documentation
   - Asset organization and file structure setup
   - Design token implementation and naming conventions
   - Cross-platform adaptation and responsive planning

4. **Deliver** — exports, usage guides, and dev-ready handoffs.
   - Final asset production and optimization
   - Developer handoff and technical documentation
   - Implementation support and design QA
   - Training materials and team onboarding
   - Launch coordination and deployment assistance

5. **Support** — ongoing asset creation and design ops.
   - Design system maintenance and updates
   - New asset creation and brand evolution
   - Performance monitoring and optimization
   - Design ops workflow improvement
   - Long-term partnership and strategic guidance
